Classroom Demos

A project site for small class demos. Click a demo to open it in a new tab.

Available demos

  • Demo 1 — Plain text page
  • Demo 2 — Example with CSS

How to update

  1. Open a file in this repo on GitHub and click the pencil icon to edit.
  2. Commit the change to the main branch. Pages will redeploy automatically.
  3. To add a new demo: click Add file → Create new file → name it demos/demo-3.html, paste content, commit.

Tip: To show a plain text file as HTML, create a demo page and paste the text inside a <pre> or <article> element (see Demo 1).

Published with GitHub Pages — update files here to change content. Site will appear at https://MEQuanWork.github.io/classroom-demos/ after you enable Pages.
